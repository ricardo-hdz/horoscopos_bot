# Horoscopos Bot
A Telegram Bot to provide you the daily horoscope (spanish only)

## Installation & Run

```
git clone
npm install
claudia create --region us-east-1 --api-module bot
claudia update --configure-telegram-bot
```

## To update bot

```
claudia update
```
